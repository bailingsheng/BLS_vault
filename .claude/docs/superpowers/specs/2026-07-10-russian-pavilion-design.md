# 俄罗斯国家馆宣传前端 — 设计规格书

**日期：** 2026-07-10
**类型：** 单页宣传展示型前端
**定位：** 进博会风格 · 俄罗斯国家馆 · 全品类商品目录

---

## 1. 概览

### 1.1 目标

为俄罗斯国家馆（进博会场景）制作一个商品展示宣传前端页面。用户可浏览全品类俄罗斯商品，按分类筛选和关键词搜索，了解商品基本信息。纯展示型，不涉及下单交易。

### 1.2 用户画像

- 进博会参观者、采购商、对俄罗斯商品感兴趣的消费者
- 中文为主要语言

---

## 2. 视觉风格

### 2.1 主题

**经典俄式奢华** — 深红底色 + 金色点缀，华丽庄重的克林姆林宫展厅感。

### 2.2 色彩方案

| 用途 | 色值 | 说明 |
|---|---|---|
| 主背景 | `#0a0a0a` / `#0d0d0d` | 深黑底色，衬托内容 |
| 品牌红 | `#8B0000` → `#4a0000` | 渐变，用于 Hero 区 |
| 金色主色 | `#DAA520` | 边框、标题、高亮 |
| 金色浅色 | `#FFD700` | 装饰文字、副标题 |
| 卡片背景 | `#141414` / `#1c1c1c` | 商品卡片底色 |
| 边框 | `#2a2a2a` / `#333` | 卡片和分隔线 |
| 文字主色 | `#ccc` / `#ddd` | 主要内容文字 |
| 文字辅助 | `#666` / `#999` | 次要说明文字 |

### 2.3 字体

- 中文标题/正文：系统默认中文字体（微软雅黑 / PingFang SC）
- 俄文/英文装饰：系统默认字体，用于品牌名 РОССИЯ
- 数字：等宽数字，用于商品规格

### 2.4 装饰元素

- Hero 区使用俄式菱形 ✦ 暗纹装饰
- 金色细线分隔重要区块
- 卡片悬停时金色边框光晕效果
- 分类标签选中态使用金色填充

---

## 3. 页面结构

自上而下 7 个区块：

### ① 顶部导航栏

- 左侧：俄罗斯国旗图标 + "РОССИЯ 俄罗斯国家馆"
- 右侧导航链接：首页、商品中心、关于我们
- 固定于页面顶部，滚动时保持可见
- 背景：半透明深黑 + 模糊

### ② 品牌 Hero 区

- 深红到暗红渐变背景
- 中央金色双线边框包裹标题：
  - 中文："俄罗斯国家馆"
  - 英文/俄文："RUSSIAN NATIONAL PAVILION" / "РОССИЯ"
- 副标题："品味纯正俄罗斯 · 从这里开始"
- 背景带俄式菱形 ✦ 暗纹装饰

### ③ 搜索区

- 居中搜索输入框，圆角 pill 样式，暗底白字
- placeholder："搜索俄罗斯商品..."
- 搜索框下方：热门搜索标签（蜂蜜、伏特加、套娃等），点击快速搜索

### ④ 分类筛选栏

- 水平排列的分类标签按钮：
  - 全部 | 食品酒水 | 工艺礼品 | 母婴个护 | 粮油农产品
- 选中态：金色填充 + 深色文字
- 未选中态：暗色背景 + 灰色文字 + 细边框
- 点击切换商品网格显示

### ⑤ 商品展示网格

- 响应式网格：
  - PC（≥1024px）：4 列
  - 平板（768-1023px）：3 列
  - 手机（<768px）：2 列
- 每张商品卡片含：
  - 商品图片区（占位 emoji 或渐变色块）
  - 商品名称
  - 规格/描述文字
- 悬停效果：卡片轻微上浮 + 金色边框光晕 + 阴影增强
- 搜索/分类切换时平滑过渡

### ⑥ 服务保障区

- 4 个信任要素图标横向排列：
  - ✈️ 源头直采 — 俄罗斯原装进口
  - ✅ 正品保证 — 官方授权认证
  - 🚚 全国配送 — 顺丰冷链直达
  - 🛡️ 无忧售后 — 7天无理由退换

### ⑦ 底部信息栏

- 品牌名 + 版权信息
- 浅色文字，低调处理
- "© 2026 Russian National Pavilion · 进博会官方授权"

## 4. 交互行为

### 4.1 搜索

- 输入时实时过滤商品（按名称 + 描述匹配）
- 无匹配结果时显示空状态："未找到相关商品"
- 支持点击热门标签快速填充搜索词

### 4.2 分类筛选

- 点击分类标签，仅显示对应分类商品
- "全部" 显示所有商品
- 搜索和分类筛选可同时生效（取交集）

### 4.3 悬停效果

- 商品卡片 hover：`transform: translateY(-4px)` + 金色边框过渡
- 分类标签 hover：边框变亮
- 导航链接 hover：文字变金色
- 所有过渡 0.3s ease

### 4.4 滚动行为

- 导航栏 固定定位
- 平滑滚动到各区块（可选）

## 5. 数据模型

### 商品数据结构

```js
{
  id: number,
  name: string,        // 商品名称
  category: string,    // 分类: '食品酒水' | '工艺礼品' | '母婴个护' | '粮油农产品'
  description: string, // 简短规格描述
  emoji: string,       // 占位图标（后续替换为真实图片 URL）
  tags: string[]       // 搜索关键词标签
}
```

### 分类

| 分类 | 标识 | 示例商品 |
|---|---|---|
| 食品酒水 | food | 巴什基尔蜂蜜、爱莲娜巧克力、标准伏特加、鱼子酱、糖果 |
| 工艺礼品 | craft | 手工套娃、琥珀吊坠、漆器盒子、巴甫洛沃披肩 |
| 母婴个护 | care | 天鹅绒护手霜、婴儿配方奶粉、沙棘面霜、天然肥皂 |
| 粮油农产品 | farm | 马克发面粉、葵花籽油、奶油奶酪、冷冻海产品 |

## 6. 技术方案

### 6.1 技术栈

- **纯静态页面**：HTML5 + CSS3 + Vanilla JavaScript
- 无框架、无构建工具、无外部依赖
- 单文件自包含（内联 CSS + JS），直接双击 `index.html` 即可浏览

### 6.2 文件结构

```
russian-pavilion/
└── index.html    # 单文件，含内联 CSS 和 JS
```

### 6.3 浏览器兼容

- Chrome、Edge、Firefox、Safari 最新版本
- 使用 CSS Grid / Flexbox（标准属性，无需前缀）
- 不使用任何非标准 API
